## 2. 操作テスト実施（R6 主導）

### 2.1 方法
- ツール: Playwright 1.56.1 + headless Chromium（`/opt/pw-browsers`）
- 端末条件: 390×844 / DPR3 / モバイル・タッチ有効（スマホ縦想定）
- スクリプト: `test-artifacts/optest.js`、成果物: `test-artifacts/baseline/*.png` + `console.log`
- フロー: ホーム→ルール→名前（空入力エラー含む）→カウントダウン→対戦（タップで選択/注ぎ）→リタイア→結果

### 2.2 結果サマリ
- 画面遷移・選択・注ぎ・リタイア・結果表示は**動作する**（HUD: time/moves/status 更新を確認）。
- **重大バグを1件検出**（下記 I-01）。
- ネットワークは当テスト環境のTLS制約で Supabase へ到達不可（`ERR_CERT_AUTHORITY_INVALID`）。本番固有ではないが、**タイムアウト/リトライ未実装**という弱点を露呈。

### 2.3 検出した不具合・課題（生ログ要約）
- `[pageerror] Cannot read properties of undefined (reading 'length')` … ロード直後に発生。
- ホームの統計テキストが初期文言のまま（`fetchPlayStats` が未実行）。
- ランキング取得失敗時は「ランキングを取得できません」にフォールバック（挙動は妥当）。

## 3. 課題・改善点の洗い出し（役割別）

### R6 QA（バグ）
- **I-01 [重大]** `drawBottle` がホーム画面で空の `state.bottles[i]`（=undefined）の `.length` を参照し例外。`initApp` の `draw()` が中断し **`fetchPlayStats()` が実行されない**。毎ロードで未捕捉エラー。

### R4 パフォーマンス／通信
- **I-02** `sbRpc`/`fetch` にタイムアウトが無い → 遅延時に「送信中…」のまま固着し得る。
- **I-03** 送信失敗時の**再試行導線が無い**（一度失敗すると諦めるしかない）。
- **I-04** 送信中の視覚的フィードバック（スピナー等）が無い。

### R2 カラー＆アクセシビリティ
- **I-05** 色のみで層を区別。赤(`#ff667a`)/緑(`#55d68b`)は色覚多様性で弁別困難。**色以外の手がかり（形）が無い**＝色合わせゲームとして致命的になり得る。

### R1 ビジュアルデザイン
- **I-06** 色層が完全フラットで質感が乏しい。ビンのガラス感も弱い。
- **I-07** HUDの時間ピルが小さく数値が読みにくい・桁変動で幅が揺れる。

### R3 操作性
- **I-08** 選択中の表示が細い橙枠のみで弱い（誤操作・見落とし）。
- **I-09** 「1手戻す」が常時有効に見え、戻せない時もタップ→エラー文言。状態が伝わらない。

### R5 UX
- **I-10** クリア時の達成演出が無く手応えが薄い。
- **I-11** 共有/コピー実行後に成否のフィードバックが無い（押せたか不明）。
- **I-12** 有効手が無くなった（詰み）ときの案内が無い。
- **I-13** a11y: `status` の更新が支援技術へ通知されない、名前入力に `label`/`aria` 紐付けが無い。

## 5. 実装計画（採用項目）

すべて `index.html` 内で完結。ロジック（ルール/タイマー/クリア判定/Undo/ランキング送信値）は変更しない。

- **F1 (P0/I-01)**: `drawBottle` の `state.bottles[i]` を `||[]` でガード。ロード時例外を解消し `fetchPlayStats` を復活。
- **F2 (P1/I-02〜04)**: `sbRpc` に `AbortController` タイムアウト(8s)。`submitScoreOnce` を最大2回試行(900ms待機)、送信中スピナー、失敗時「再送信する」ボタンを表示。
- **F3 (P1/I-05)**: 各色に固有マーカー `red:▲ / blue:● / yellow:■ / green:◆` を層中央へ描画。黄は暗色、他は白でコントラスト確保。
- **F4 (P1/I-06)**: 色層を縦グラデ＋上部ハイライト、ビンに縦の光沢ライン。ガラス内クリップで内容を整える。
- **F5 (P1/I-08,09)**: 選択ビンを約7px持ち上げ＋外側グロー＋影。`updateHud` で「PLAYING かつ undo可」以外は Undoボタンを `disabled`。
- **F6 (P2/I-07)**: 時間表示を `tabular-nums`、ピル折返し抑止。
- **F7 (P2/I-10,11)**: クリア時に独立オーバーレイ canvas の紙吹雪（約1.6s）。共有/コピー後にトースト表示。
- **F8 (P2/I-12)**: `hasAnyMove()` を追加し、注ぎ後に有効手が無ければヒント文言へ。
- **F9 (P2/I-13)**: `status` に `aria-live="polite"`、名前入力に `aria-label`、トーストに `role="status"`。
